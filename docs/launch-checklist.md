# Launch Checklist

## Pre-Launch (Before First Post)

### Infrastructure
- [ ] n8n running locally via Docker
- [ ] Cloudflare Tunnel or ngrok configured for webhooks
- [ ] All API keys obtained and tested (`scripts/test-apis.sh` passes)
- [ ] Telegram bot created and tested
- [ ] Buffer or scheduling tool connected

### Character
- [ ] Base character references generated (front, 3/4, full body)
- [ ] 5 outfit variations generated and saved
- [ ] 5 setting references generated
- [ ] Character consistency verified across 10+ test images
- [ ] ElevenLabs voice cloned and configured

### Social Profiles
- [ ] Instagram account created
  - Username: @{choose}
  - Bio includes: "AI-generated content | Virtual Creator"
  - Bio includes: link-in-bio URL
  - Profile photo: character headshot
- [ ] TikTok account created
  - Bio includes AI disclosure
  - Profile photo matches Instagram
- [ ] YouTube channel created
  - Channel description includes AI disclosure
  - Profile and banner art uploaded
- [ ] X/Twitter account created
  - Bio includes AI disclosure
  - Profile photo matches other platforms

### Content
- [ ] 12 Instagram grid posts generated and approved
- [ ] 5 TikTok videos generated and approved
- [ ] 3 YouTube Shorts generated and approved
- [ ] First X thread written
- [ ] Content calendar for first month planned

### Legal
- [ ] AI disclosure in all bios: "AI-generated content | Virtual Creator"
- [ ] Disclosure language for sponsored posts defined (#ad, #sponsored)
- [ ] EU AI Act compliance reviewed
- [ ] Swedish Marknadsforingslagen compliance reviewed
- [ ] FTC guidelines reviewed

## Post-Launch (First Week)

- [ ] Review first analytics report
- [ ] Adjust posting times if engagement data suggests better windows
- [ ] Respond to any comments/engagement manually to seed initial community
- [ ] Verify character consistency across all published content
- [ ] Check that all disclosure requirements are met on each post
- [ ] Document any workflow issues for iteration
- [ ] Plan Week 2 content based on Week 1 performance
