# Typography Guidelines

## Primary Font
**Inter** — Clean, modern sans-serif. Used for all body text, captions, and UI elements.

## Heading Font
**Space Grotesk** — Geometric, tech-forward. Used for titles, hooks, and text overlays on images.

## Monospace (Code/Tech Content)
**JetBrains Mono** — For any code snippets or technical content in posts.

## Usage Rules
- Headlines: Space Grotesk Bold, 24-36pt
- Body text: Inter Regular, 14-16pt
- Captions: Inter Regular, 14pt
- Text overlays on images: Space Grotesk Semi-Bold, white with subtle shadow
- Hashtags: Inter Regular, 12pt

## Text Overlay Style
- Always use white text on darkened/blurred background area
- Max 2 lines of text per overlay
- Left-aligned, lower-third positioning preferred
